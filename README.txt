INTRODUCTION
------------

Provides a reaction plugin for the Context module that allows the disabling of
blocks.


INSTALLATION
------------

 * Install as usual, see http://drupal.org/documentation/install/modules-themes/modules-7 for further information.

CONFIGURATION
-------------

No special configuration necessary beyond the Context reaction UI.
